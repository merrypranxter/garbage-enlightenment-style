# SYMBOL BLEED NOTES

This file focuses on **symbol bleed** in Garbage Enlightenment.

Symbol bleed is what happens when one symbol starts leaking into another symbol’s job,
shape, or emotional role.

Examples:
- a heart starts behaving like a mouth
- a rainbow starts behaving like body matter
- an eye starts behaving like a warning sign
- a portal starts behaving like a face
- a star starts behaving like a watcher

This is one of the style’s deepest structural laws.

## Core principle

A symbol should not always remain sealed inside its original meaning.

In Garbage Enlightenment, symbols often:
- contaminate each other
- exchange roles
- overlap semantically
- partially fuse

- remain legible while becoming unstable

That instability is where a lot of the electricity lives.

## WHY SYMBOL BLEED MATTERS

Without symbol bleed, the style can become:
- too literal
- too modular in a dead way
- too neat
- too decorative
- too mascot-stable
- too spiritually obvious

Symbol bleed creates:
- mutational pressure
- LSD-logic adjacency
- emotional instability
- deeper motif reuse
- stronger hybrid icons

## MAIN SYMBOL BLEED TYPES

---

## 1. SHAPE BLEED

### Description
One symbol’s contour begins to support another symbol’s reading.

### Examples
- heart outline becoming lips
- star points becoming lashes
- halo ring becoming portal mouth
- blob shape becoming head

### Effect
- strong visual fusion
- good portability
- useful for icons and loops

---

## 2. ROLE BLEED

### Description
The symbol keeps its shape but changes what job it is doing.

### Examples
- eye as warning sign
- heart as surveillance icon
- hand as portal
- rainbow as anatomy

### Effect
- semantic instability
- symbolic over-meaning
- stronger fake revelation

---

## 3. TONE BLEED

### Description
One symbol imports its emotional role into another.

### Examples
- a cute star gaining menace
- a skull gaining sweetness
- a heart gaining watchfulness
- a mouth gaining devotional framing

### Effect
- emotional cross-signal
- richer icon tone
- stronger Garbage Enlightenment flavor

---

## 4. FIELD BLEED

### Description
A symbol starts to dissolve into a field or emerge from one while keeping some identity.

### Examples
- eye into contour bands
- mouth from rainbow field
- hand from thermal map
- face from sigil swarm

### Effect
- figure-pattern bridge
- abstract subsystem integration
- stronger motion logic

## GOOD SYMBOL BLEED QUALITIES

- readable but unstable
- simple but over-significant
- not fully collapsed into mush
- strong enough to notice quickly
- weird enough to keep mutating mentally

## BAD SYMBOL BLEED QUALITIES

- random mashup with no contour logic
- unreadable hybrid sludge
- empty surrealism
- over-polished hybrid logos
- generic “fusion” with no tonal or symbolic pressure

## LSD-LOGIC NOTE

Symbol bleed is one of the clearest expressions of **cartoonized LSD-hallucination logic**
because it allows:
- one thing to be several things
- meaning to move faster than form
- symbolic boundaries to soften without disappearing
- visual over-meaning through simple hybrids

This should be treated as a core style engine, not a side effect.

## PROMPT PHRASES

Useful phrases:
- symbolic role bleed
- one symbol leaking into another
- hybrid sign under mutational pressure
- decorative form with semantic instability
- body-symbol contamination event
- readable but bleeding symbolic identity

## FINAL RULE

A good symbol bleed event in Garbage Enlightenment should feel like:
**one icon quietly stealing another icon’s job while they both pretend this is normal**
