# SIGIL SWARM NOTES

This file focuses on **sigil swarms** in Garbage Enlightenment.

A sigil swarm happens when many small fake-symbolic marks begin to behave like:
- a field
- a chant
- a ritual cloud
- a decorative bureaucracy
- a pseudo-language
- distributed symbolic pressure

This is useful because it lets the style create density without losing thematic coherence.

## Core principle

A sigil swarm should not feel like:
- elegant sacred geometry
- polished ceremonial seal design
- neat occult wallpaper

- arbitrary doodle spam

It should feel like:
- fake ritual repetition
- low-budget symbolic insistence
- body-contaminated sign language
- marks that are trying too hard to matter
- decorative marks becoming a system before they deserve one

## WHY SIGIL SWARMS WORK

They create:
- pattern consciousness
- symbolic clutter with pressure
- ritual bureaucracy
- fake revelation atmosphere
- over-meaning through repetition
- a sense that the image belongs to a larger visual cult language

### Description

## GOOD SIGIL SWARM QUALITIES

- repeated with variation
- visually related but not identical
- slightly asymmetrical
- slightly rude
- able to support larger icons
- emotionally active
- not too polished

## BAD SIGIL SWARM QUALITIES

- perfect symmetry
- ornamental purity
- empty occult texture
- random scratch marks with no internal family logic
- luxury branding energy

## LSD-LOGIC NOTE

Sigil swarms strongly support **cartoonized LSD-hallucination logic** when:
- small repeated marks start feeling over-significant
- pattern becomes pseudo-language
- symbolic meaning multiplies through swarm behavior
- the field feels like it is trying to say something in an unstable visual code

That is a major strength and should be leaned into.
