# EYE SWARM NOTES

This file focuses on eye swarms in Garbage Enlightenment.

Eye swarms are one of the strongest motif systems in the whole style because they combine:
- body glyph logic
- surveillance pressure
- symbolic repetition
- pattern consciousness
- fake revelation
- decorative overload
- LSD-logic over-meaning

## Core principle

An eye swarm should not feel like:
- generic “creepy eyes everywhere”
- horror cliché

- random occult wallpaper
- decorative repetition only

It should feel like:
- distributed watchfulness
- symbolic chorus
- over-significant ornament
- pattern becoming conscious
- fake sacred surveillance
- body signs repeating until they become a field

## MAIN EYE SWARM MODES

---

## 1. DECORATIVE EYE SWARM

### Description
Small eyes repeat around a main symbol or creature.

### Effect
- contamination
- pressure
- Sweet Corruption support

- soft menace

### Best use
- posters
- sheets
- icons
- revelation framing

---

## 2. FIELD EYE SWARM

### Description
Eyes emerge throughout an abstract field.

### Effect
- field consciousness
- face pressure
- LSD-logic pattern awareness
- body-haunted abstraction

### Best use
- contour fields
- shader prompts

- thermal emergence

---

## 3. CREATURE EYE SWARM

### Description
Many little eye motifs behave like:
- distributed creaturehood
- swarm heads
- eye-born beings
- watchful little organisms

### Effect
- cute menace
- body goblin swarm energy
- pattern-to-creature bridge

---

## 4. REVELATION EYE SWARM

### Description
The eyes are arranged like:

- witness icons
- halo orbit
- cosmic approval system
- fake sacred bureaucracy

### Effect
- ritual bureaucracy
- mock revelation overload
- cheap transcendence pressure

## GOOD EYE SWARM QUALITIES

- repeated with variation
- not perfectly uniform
- some scale shifts
- some eyes more intense than others
- enough spacing to remain readable
- enough clustering to feel like pressure

## BAD EYE SWARM QUALITIES

- pure horror spam
- flat wallpaper repetition
- dead symmetry

- generic trippy eye textures
- over-polished mystical repeating eye art

## EYE SWARM + LSD-LOGIC

This is one of the repo’s cleanest uses of **cartoonized LSD-hallucination logic** because:
- repetition creates over-meaning
- simple eyes start acting like distributed consciousness
- pattern becomes watchfulness
- significance multiplies through swarm behavior

This should be reused constantly.
