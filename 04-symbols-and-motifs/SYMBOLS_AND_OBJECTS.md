# SYMBOLS AND OBJECTS

This file covers non-body symbols, symbolic objects, and visual props that commonly belong in
Garbage Enlightenment.

These are not neutral props.
They are part of the style’s symbolic ecosystem.

## Core principle

Objects should usually feel:
- slightly charged
- slightly stupid
- slightly significant
- available for mutation
- compatible with both cute and bodily systems

- able to function as decoration and symbol at once

Objects in Garbage Enlightenment should rarely feel neutral.

They should feel like:
- signs
- warnings
- jokes
- relics
- fake sacred tools
- decorative contaminants
- emotionally overcommitted symbols
