# MOTIF LIBRARY
This file lists recurring motifs that belong naturally inside Garbage Enlightenment.
These motifs are not all equal in every image.
They are ingredients, not obligations.
But together they define the symbolic ecosystem of the style.
## Core principle
A motif in this repo should usually be able to function as more than one thing.
It can be:
- subject
- symbol
- decoration
- emotional cue
- mutation site
- animation node
- joke
- icon
- portal
- recurring visual chorus
