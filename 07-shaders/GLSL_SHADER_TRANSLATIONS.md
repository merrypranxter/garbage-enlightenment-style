# GLSL SHADER TRANSLATIONS
This document translates the abstract and metamorphic logic of Garbage Enlightenment into
terms useful for shaders, procedural systems, motion systems, and generative image engines.
This is not about making the shader “look cool” in a generic psychedelic sense.
It is about making the shader behave like the style.
## Core principle
Garbage Enlightenment shader logic should support:
- symbolic emergence
- morphable contour boundaries
- cheap hallucination fields
- figure–pattern permeability
- low-fi psychedelic pressure
- body-glyph reveal potential
- fast intermediate-state meaning
The shader should feel like:
- a substrate for mutation
- a soup of almost-symbols
- a fake sacred signal field
- a cartoonized hallucination engine
## Main shader families
--## 1. CONTOUR BAND SHADERS

### Goal
Create topographic, wavy, separated color zones that can:
- frame silhouettes
- imply emergence
- act like psychedelic map-space
- support eye/hand/face reveal
### Useful techniques
- fbm-based scalar fields
- thresholded noise
- contour quantization
- domain warping
- isoline extraction
- posterized gradient bands
### Best use
- hand silhouettes
- eye icons
- aura backgrounds
- figure–field transitions
- halo-stage imagery
--## 2. LIQUID HALLUCINATION FIELDS
### Goal
Make moving fields that feel like:
- oil slicks
- thermal maps
- false depth
- chemical rainbows
- synthetic pressure regions
### Useful techniques
- animated fbm
- multi-frequency domain warp
- cosine palette mapping
- turbulence
- time-varying gradient remapping
- threshold bleed between bands
### Best use
- portal interiors

- abstract backgrounds
- atmospheric signal soup
- pattern-to-figure emergence zones
--## 3. THRESHOLD BLOB SYSTEMS
### Goal
Generate blob regions that can:
- merge
- split
- bulge
- reveal figure-like cavities
- support creature substrate
### Useful techniques
- metaballs
- signed distance field blending
- smooth minimum operations
- threshold masks
- region expansion / contraction
- cellular noise
### Best use
- body-blob creatures
- cavity emergence
- soft anatomy maps
- symbolic lump logic
- pre-figurative mutation space
--## 4. FIGURE–PATTERN BRIDGE SHADERS
### Goal
Allow abstract fields to resolve into readable symbolic units.
### Useful techniques
- mask interpolation
- silhouette reveal by threshold pass
- field-guided contour extraction
- texture-space morph between shape libraries
- edge detection on animated scalar fields

- signed distance transitions
### Best use
- eye emerging from plasma
- mouth cavity from contour collapse
- hand symbol appearing in hallucinatory field
- face half-resolved from abstract motion
This is one of the most important shader categories in the whole repo.
--## 5. CHEAP DIGITAL DAMAGE LAYERS
### Goal
Add low-fi synthetic friction without turning the image into generic glitch art.
### Useful techniques
- subtle pixel swim
- dither
- scanline suggestion
- compression-like noise
- low-res mask wobble
- threshold shimmer
- frame-lag trails
- quantization artifacts
### Best use
- reinforcing cheapness
- breaking over-smoothness
- emphasizing synthetic image identity
- animation roughness
--## 6. BODY-GLYPH REVEAL SYSTEMS
### Goal
Procedurally birth symbolic body units out of field activity.
### Reveals to support
- eye
- mouth
- teeth

- tongue
- hand
- skull
- face silhouette
- creature profile
### Useful techniques
- shape masks blended into noise fields
- contour-guided pattern revelation
- multi-stage threshold emergence
- template interpolation
- signed distance shape blending
- cellular-to-symbol transition logic
--## 7. FAST INTERMEDIATE-STATE SHADERS
This style cares deeply about half-formed states.
### Important behavior
The shader should not only optimize for:
- clean reveal
- final crisp icon
- stable finished frame
It should also produce:
- semi-recognizable forms
- flickering symbolic almost-states
- unstable partial emergence
- contour confusion that still feels intentional
This is key to the repo’s resemblance to **cartoonized LSD-hallucination logic**:
- forms over-signal meaning
- one form contains another
- identity flickers quickly
- transition states feel alive, not disposable
### Useful techniques
- noisy interpolation
- partially thresholded masks
- edge-smear transition zones
- time-offset blended layers
- contour lag

- feedback decay
- palette region mismatch during morph

## Color guidance for shaders
Base shader palettes should favor:
- acid yellows
- hot pinks
- electric cyan
- weird peach flesh
- toxic greens
- cartoon reds
- cheap blues
- black contour boundaries
- occasional muddy support tones
But palette systems should remain modular so later variants can be added.
## Things to avoid in shader translation
### Bad drift
- generic rainbow wallpaper
- too-smooth cosmic gradient art
- majestic nebula vibes
- polished visionary tunnel effects
- clean music visualizer aesthetics
- sterile procedural beauty with no symbolic threat or body weirdness
### Correction
Add:
- thresholds
- contour conflict
- shape reveals
- body glyph hints
- awkwardness
- pressure
- partial wrongness
## System design rule
A shader in this style should not merely generate pattern.
It should generate:
- almost-creatures

- almost-symbols
- almost-portals
- body-adjacent abstractions
- unstable readable forms
The field should always feel like it might cough up an eye.

plasma fields are always on the verge of becoming something embarrassingly meaningful**
