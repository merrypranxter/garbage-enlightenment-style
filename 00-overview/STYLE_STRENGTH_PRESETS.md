# STYLE STRENGTH PRESETS

This file defines practical strength presets for using Garbage Enlightenment at different
intensities.

These are useful for:
- prompting
- app controls
- style-transfer systems
- staged generation
- testing how much of the repo identity is needed for a given task

## Core principle

Not every output needs full repo saturation.
But lower-strength outputs should still retain enough DNA to be recognizable.

## PRESET 1 — LIGHT INFUSION

### Use for
- gentle flavoring
- mixing with another primary style
- subtle stylistic hinting

### Main traits
- one or two body glyphs
- mild Sweet Corruption
- slight anti-polish linework
- low-to-medium symbolic charge
- limited mutational pressure

### Prompt language
- slight Garbage Enlightenment influence
- subtle body-glyph symbolism
- mild sweet contamination
- restrained fake revelation energy

### Risk
Can become too generic if under-specified.

## PRESET 2 — CLEAR MODE

### Use for
- most standard outputs
- strong but not overwhelming style application
- reusable prompt backbone

### Main traits
- clear body glyph anchor
- clear tone friction
- some Sweet Corruption or mock esoterica
- rude black outlines
- synthetic color
- implied mutation

### Prompt language
- Garbage Enlightenment style
- body-glyph symbolism
- Sweet Corruption or fake revelation
- rude black outlines
- symbolic instability

### Risk
May still drift if anti-drift phrases are missing.

## PRESET 4 — FULL FREAK SATURATION

### Use for
- extreme outputs
- motion-heavy work
- highly symbolic images
- maximal identity testing

### Main traits
- multiple body glyphs
- rapid mutational pressure
- abstract/figurative bleed
- strong fake revelation
- tonal instability
- rich contamination

- high LSD-logic emphasis
- lots of semi-states or implied semi-states

### Prompt language
- full freak Garbage Enlightenment
- body-haunted hallucination logic
- rapid symbolic reassignment
- one form implying several forms
- fake revelation and Sweet Corruption coexisting
- rude cartoon metamorphosis
- no polish, no generic trippy drift

### Risk
Can become too noisy without a strong subject anchor.

### Use for

- transition sequences

### Main traits

- abstract hallucination field emphasis
- contour-band logic
- emergence pressure

## PRESET 6 — SWEET CORRUPTION HEAVY

### Use for
- cute branch work
- sticker sheets
- sweeter posters
- adorable-but-wrong outputs

### Main traits
- stars/hearts/rainbows prominent
- cute creatures or motifs
- contamination kept active
- lower horror
- strong tone friction between sweetness and wrongness

### Prompt language
- Sweet Corruption heavy
- cute symbols contaminated by body glyphs
- toy-like but weird
- sweet and symbolically wrong

### Risk
Can drift into pristine cuteness without body sabotage.

## FINAL RULE

The stronger the preset, the more important it becomes to keep:
- one clear symbolic anchor
- motif intelligence
- anti-drift controls
- compositional focus

Strength should increase identity, not just clutter.
