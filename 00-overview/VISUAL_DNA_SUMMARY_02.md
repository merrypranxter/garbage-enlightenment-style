# VISUAL DNA SUMMARY 02

This file is a second compressed summary of Garbage Enlightenment, written in a more
system-friendly and curation-friendly form.

Use this when:
- handing the style to another AI system
- explaining the repo fast
- building internal notes
- summarizing for app logic
- creating onboarding docs

## CORE IDENTITY IN ONE BLOCK

Garbage Enlightenment is a low-fi, body-haunted, cute-contaminated, mock-esoteric
psychedelic cartoon language built from rude black outlines, simplified symbolic forms, body
parts used as glyphs, fake revelation imagery, abstract hallucination fields, and fast or implied
metamorphosis. It often resembles cartoonized LSD-hallucination structure through rapid
symbolic reassignment, figure-pattern instability, significance overload, and one form implying
several others at once, but it translates that logic into cheap, anti-polish, emotionally unstable
cartoon form rather than polished visionary art.

## SIX FAST ANCHORS

1. **Body glyphs**
- eyes, mouths, hands, heads, skulls, blobs

2. **Sweet Corruption**
- stars, hearts, rainbows, cute motifs contaminated by wrongness

3. **Mock Esoterica**
- eye pyramids, halos, portals, fake sacred warnings, parody ritual charge

4. **Cheap Hallucination Fields**
- contour bands, thermal maps, plasma blobs, emergence substrate

5. **Mutational Pressure**
- images feel paused mid-becoming, not fully settled

6. **Anti-Polish Tone**

- rude, low-fi, slightly embarrassing, synthetic, alive

## WHAT MAKES IT FEEL LIKE ITSELF

The style becomes most recognizable when:
- the image is readable but unstable
- a body part is doing symbolic work
- something cute has been contaminated
- something sacred-looking has been made slightly ridiculous
- abstraction feels like it could become a creature or glyph
- the image feels emotionally mixed rather than pure
- the whole thing seems like it might morph if you stare at it long enough

## TONE PROFILE

The style can move between:
- cute absurdity
- freak tenderness
- gross-funny body weirdness
- fake revelation
- goofy cosmic dread
- chemical wonder

Best results usually mix at least two.

## LSD-LOGIC NOTE

When this repo references LSD-adjacent qualities, it means:
- unstable significance
- recursive implication
- one shape containing several possible readings
- fast visual reassignment
- pattern becoming body, body becoming symbol, symbol becoming creature

It does **not** mean:
- documentary psychedelic realism
- just adding more rainbows
- default “trippy” clichés

## FINAL MINI-SUMMARY

Garbage Enlightenment is:
**a rude, symbolic, sweet-contaminated, fake-sacred cartoon mutation language that turns
hallucination logic into sticker-like body weirdness**
