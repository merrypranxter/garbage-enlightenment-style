# LINEWORK AND SIMPLICITY
Garbage Enlightenment depends heavily on graphic clarity.
Its weirdness becomes stronger, not weaker, when the drawing language stays relatively simple.
This style does not need virtuoso rendering.
It needs rude intelligence.
## Core principle
The line should:
- contain
- clarify
- symbolize
- cheapen
- unify
- slightly disrespect the image
The line should not:
- over-polish
- over-describe
- become precious
- become ornamental for its own sake
## Main line qualities
### Common traits
- black or very dark outline
- thin-to-medium weight
- direct shape containment
- simple edge logic
- occasional wobble or hand-done roughness
- cartoon confidence without over-refinement

### Emotional effect
Good linework can make the image feel:
- low-fi
- bodily
- toy-like
- iconic
- funny
- slightly cheap
- mutation-ready
## What line is doing in this style
Line is not just decorative.
It often performs several jobs at once:
- separates symbolic units
- keeps mutation legible
- supports sticker/emblem readability
- strengthens flatness
- makes weird anatomy easier to accept
- protects the image from drifting into painterly slop
- helps the composition survive color chaos
## Simplicity as force multiplier
The style often gets stronger when:
- forms are simplified
- anatomy is reduced
- face logic is compressed
- creatures are built from few clear units
- symbols are stated bluntly
This is because simplification makes room for:
- substitution
- visual humor
- symbolic charge
- animation logic
- repeated motifs
- dream readability
## Good simplicity
Good simplicity is:

- intentional
- legible
- graphic
- rude
- compressed
- transformable
It is not:
- lazy emptiness
- flavorless minimalism
- dead vector neatness
- underdesigned blankness
## Line modes
--## 1. CONTAINMENT LINE
Used to hold:
- body glyphs
- creature silhouettes
- heads
- hands
- decorative motifs
- contour-relevant shapes
This is the most basic line mode.
--## 2. SYMBOL LINE
Used when the line itself makes the image read as:
- emblem
- icon
- ritual sign
- sticker
- warning
- parody symbol
This is especially important in mock-esoteric mode.
---

## 3. MUTATION LINE
A line that implies:
- stretching
- leaking
- soft instability
- contour inheritance
- shape continuation
This is crucial for animation-aware design and still images that imply morphability.
--## 4. CHEAPNESS LINE
The line can also make the image feel:
- bootleg
- low-budget
- anti-luxury
- unserious in a good way
- raw enough to stay alive
This is one of the strongest anti-drift tools.
## Detail philosophy
Detail should be selective.
Strong places for detail:
- eyes
- teeth
- mouth cavities
- key decorative motifs
- symbolic focal points
- selected contour interruptions
- repeated glyph clusters
Weak places for detail:
- unnecessary anatomy realism
- excessive texture rendering
- polished shading logic
- refined surface finish

## Relation to LSD-hallucination logic
This style can nod toward **LSD-hallucination imagery**, but the line keeps it from dissolving
into formless over-complexity.
The line acts like a rude compression system:
- it turns visual overload into symbol
- it turns morphing into readable contour events
- it keeps significance from becoming mush
- it makes the hallucination logic cartoon-legible
So the line is one of the style’s main translation devices.
## Common failure modes
### Too polished
Line gets slick, perfect, design-school clean.
### Too weak
Forms lose readability and drift into soft color mush.
### Too detailed
Image becomes illustrative rather than symbolic.
### Too ornamental
Line becomes decorative filigree instead of graphic force.
### Too vector-corporate
Image starts looking like startup illustration or merch branding.
## Correction rules
If line is too polished:
- roughen edges slightly
- simplify anatomy
- reduce precision
- let a few things feel hand-wrong
If line is too weak:
- darken outline
- strengthen silhouette
- clarify major units
- reduce competing texture

If line is too boring:
- add one strange contour decision
- give one part of the figure mutational pressure
- interrupt symmetry
- let one body glyph dominate
## Final rule
The line in Garbage Enlightenment should feel like:
**a cartoon containment spell barely keeping the image from turning back into hallucination
soup**
