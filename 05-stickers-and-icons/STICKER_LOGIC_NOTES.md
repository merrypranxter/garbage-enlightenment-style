# STICKER LOGIC NOTES

This file explains sticker logic in Garbage Enlightenment.

Sticker logic is important because many of the style’s strongest images behave less like full
scenic illustrations and more like:
- stickers
- badges
- patches
- little holy warnings
- symbolic decals
- fake ritual merch
- body-sign labels

## Core principle

A Garbage Enlightenment image gets stronger when it can survive being:
- isolated
- flattened
- cut out
- repeated
- arranged in sheets
- treated like a portable symbol

That does not mean all images must literally be stickers.
It means they should often have:
- emblem clarity
- symbolic compactness
- cutout readability

- graphic force

## WHAT STICKER LOGIC DOES

Sticker logic helps:
- simplify the image
- strengthen motif identity
- keep symbolism blunt
- support repetition and swarms
- prevent over-rendered drift
- make weirdness portable
- make fake revelation feel commodified in a funny way

## CORE STICKER QUALITIES

### 1. Strong silhouette
A sticker needs to read fast.

### 2. Clear internal symbol
The core icon must hold even at small scale.

### 3. Outline containment
Black outlines or strong contour edges help a lot.

### 4. Emotional compression
The image should carry tone quickly:
- cute
- body-weird
- fake sacred
- eerie
- pathetic
- chemically overcommitted

### 5. Modular repeatability
A sticker-like image should be easy to:
- repeat
- remix
- swarm
- arrange
- recolor

## STICKER TYPES IN THIS STYLE

### Body glyph sticker
Eye, mouth, hand, skull, face, blob, etc.

### Sweet Corruption sticker
Heart-eye, rainbow-mouth, star-face, dolphin-weird, etc.

### Mock Revelation sticker
Eye pyramid, halo hand, fake sigil, warning icon, etc.

### Creature sticker
Body goblin, face-creature, cute-corrupted beast, etc.

### Abstract sticker
Contour emblem, field badge, thermal glyph, etc.

## STICKER COMPOSITION RULES

- keep one main motif dominant
- allow one or two contamination layers
- avoid too much scenic depth
- let the image feel portable
- favor icon compactness over full environment logic

## STICKER + LSD-LOGIC

Sticker logic becomes especially strong when the sticker looks:
- simple at first
but
- semantically unstable on second glance

That is a very efficient static version of cartoonized LSD-hallucination logic:
the image reads fast, but keeps changing its meaning.

## PROMPT PHRASES

Useful phrases:
- sticker-like symbolic compression
- fake sacred decal logic
- cutout emblem weirdness
- portable body glyph icon
- low-fi symbolic sticker force
- contaminated sticker-sheet energy
- body-haunted decal image

## FINAL RULE

A good Garbage Enlightenment image should often feel like:
**something that could be slapped on a notebook, a shrine, or a warning sign with equal
legitimacy**
