# CAVITY LOGIC NOTES

This file focuses on cavity logic in Garbage Enlightenment.

Cavities are one of the style’s strongest shared structures because they can function as:
- mouths
- eye sockets
- portals
- wounds
- openings
- voids
- symbolic centers
- black graphic anchors

A cavity is often where:
- body
- symbol
- threshold

- fake sacredness
- humor
- menace

all collide.

## Core principle

A cavity in Garbage Enlightenment should not feel merely like:
- dark shading
- generic hole
- realistic wound
- anatomical realism

It should feel like:
- an active symbolic opening
- a black sign in the image
- a threshold with body memory
- an emotional puncture
- a visual event

## WHY CAVITIES MATTER

Cavities provide:

- high contrast anchors
- body pressure
- portal logic
- soft menace
- fake revelation centers
- body-symbol conversion points

They are one of the most useful simple forms in the entire repo.

## MAIN CAVITY TYPES

---

## 1. MOUTH CAVITY

### Description
A black mouth interior treated as a threshold or event.

### Effect
- body weirdness
- portal logic
- gross-funny charge
- strong iconography

---

## 2. EYE CAVITY

### Description
A socket or eye opening that feels deeper than normal.

### Effect
- watchfulness
- revelation
- void pressure
- fake spiritual seriousness

---

## 3. SKULL CAVITY

### Description
Sockets, nasal cavity, or implied emptiness in skull form.

### Effect
- cute necrosis
- decorative mortality
- symbolic absence

- sweet dread

---

## 4. FIELD CAVITY

### Description
A dark region in abstraction behaving like:
- face pressure
- mouth seam
- portal opening
- emergence node

### Effect
- figure-pattern bridge
- body-haunted abstraction
- latent creaturehood

---

## 5. HEART / SYMBOL CAVITY

### Description
A decorative symbol develops a dark seam or opening.

### Effect
- Sweet Corruption
- symbolic instability
- body intrusion
- emotional complexity

## GOOD CAVITY QUALITIES

- visually blunt
- high contrast
- semantically active
- slightly excessive
- capable of multiple readings
- body-linked even when abstract

## BAD CAVITY QUALITIES

- realistic gore hole
- empty shadow with no symbolic role
- too many cavities with no hierarchy
- over-rendered depth realism
- horror grimness without play or fake sacredness

## CAVITY + LSD-LOGIC

Cavity logic becomes especially Garbage Enlightenment-specific when:
- one opening can be read several ways
- a dark void suddenly carries too much meaning
- body and abstraction organize around a simple black seam
- a decorative symbol begins to behave like a living opening

That is very strong cartoonized LSD-hallucination structure.

## PROMPT PHRASES

Useful phrases:
- symbolic cavity logic
- black opening acting like a portal
- body-haunted cavity sign
- dark seam under revelation pressure
- threshold cavity
- cartoon void with anatomy memory

## FINAL RULE

A good cavity in Garbage Enlightenment should feel like:
**a small black place where the image has started taking itself too seriously**

# ICON FAILURES

This file focuses on common icon failures in Garbage Enlightenment.

Because the style relies so heavily on:
- symbols
- stickers
- emblems
- body glyphs
- warning signs
- little fake sacred icons

icon failure can kill an image very fast.

## Core principle

A good icon in this style should feel:
- clear
- rude
- charged
- slightly unstable

- emotionally compressed
- body-aware
- portable
- fake-significant

An icon fails when it becomes:
- too clean
- too generic
- too decorative
- too empty
- too pretty
- too static
- too corporate

## MAIN ICON FAILURE TYPES

---

## 1. TOO CLEAN

### Symptoms
- perfect vector smoothness
- sterile line
- brand-ready finish

- no weird contour pressure

### Why it fails
The style needs anti-polish and low-fi life.

### Fix
Roughen slightly, contaminate, reduce perfection.

---

## 2. TOO EMPTY

### Symptoms
- simple icon but no pressure
- symbol reads instantly and then dies
- no alternate meanings
- no contamination
- no emotional weirdness

### Why it fails
The icon must do more than look neat.

### Fix
Add:

- body weirdness
- cute contamination
- fake sacred framing
- semantic instability

---

## 3. TOO PRETTY

### Symptoms
- aesthetically flattering
- balanced in an elegant way
- nice but weak
- pretty symbol with no teeth

### Why it fails
Pretty without tension collapses the style.

### Fix
Add rude color, body intrusion, asymmetry, or cavity logic.

---

## 4. TOO CORPORATE

### Symptoms
- app icon vibes
- startup mascot logo feel
- modern illustrative cleanliness
- emotionally safe

### Why it fails
Garbage Enlightenment symbols should not feel consumer-safe.

### Fix
Break the polish and increase symbolic weirdness.

---

## 5. TOO SACRED IN THE WRONG WAY

### Symptoms
- reverent mystical icon
- luxury occult badge
- holy-looking but too noble
- overclean halo logic

### Why it fails

The repo wants Cheap Holiness, not reverent elegance.

### Fix
Add body contamination, parody, and anti-luxury cues.

---

## 6. TOO GENERIC WEIRD

### Symptoms
- weird shape, weird eye, weird rainbow, but no internal family logic
- random icon slop
- no role pressure

### Why it fails
Not all weirdness is useful.

### Fix
Clarify motif family and symbolic role.

## GOOD ICON TEST

Ask:
- what is its main job?

- what is contaminating that job?
- what emotional tone does it compress?
- could it mutate into something else?
- does it feel like it belongs to a larger symbolic system?

If the answer is mostly yes, the icon is probably alive.

## LSD-LOGIC NOTE

Icons become more Garbage Enlightenment-specific when they are:
- simple in silhouette
- unstable in semantic role
- over-significant in a cartoon way
- able to imply multiple readings at once

That is one of the most useful compact versions of cartoonized LSD-hallucination structure.

## PROMPT PHRASES

Useful anti-failure phrases:
- keep the icon rude and symbolically unstable
- avoid clean logo polish
- make the emblem portable but contaminated
- preserve fake significance

- let the icon imply alternate roles
- avoid corporate icon neatness

## FINAL RULE

A failed icon feels like a graphic asset.
A successful Garbage Enlightenment icon feels like:
**a little symbolic object with a private agenda**
