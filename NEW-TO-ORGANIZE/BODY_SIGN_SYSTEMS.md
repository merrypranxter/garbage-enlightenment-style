# BODY SIGN SYSTEMS

This file explains how body parts can be organized into sign systems inside Garbage
Enlightenment.

This is useful because the style often treats anatomy as:
- icon language
- sticker language
- warning language
- revelation language

- decorative symbolic code

A body sign system is what happens when body glyphs stop behaving like isolated motifs and
start behaving like a visual alphabet.

## Core principle

A body sign system should feel like:
- a set of visual categories
- a low-budget symbolic language
- an unofficial warning code
- a fake sacred body bureaucracy
- anatomy turned into stickers and regulations

It should not feel like:
- medical diagramming
- realistic anatomy instruction
- clinical body charts

- polished icon UI

## BUILDING A BODY SIGN SYSTEM

### Step 1
Choose 3–5 main sign classes.

### Step 2
Assign recurring motif families to each.

### Step 3
Give each class:

- visual frame
- tone
- contamination level
- optional cute support motifs
- optional fake sacred support motifs

### Step 4
Repeat them with variation.

This creates a believable internal language.

## LSD-LOGIC NOTE

Body sign systems become especially on-brand when they feel like:
- anatomy has become administrative
- body parts carry too much symbolic weight
- categories of meaning are built from flesh logic
- the sign system seems both ridiculous and spiritually overcommitted

That is a powerful route into cartoonized LSD-hallucination structure.
