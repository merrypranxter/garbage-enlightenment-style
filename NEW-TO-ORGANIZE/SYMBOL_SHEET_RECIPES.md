# SYMBOL SHEET RECIPES

This file contains recipes for making symbol sheets, glyph packs, sticker pages, emblem arrays,
and motif collections in the Garbage Enlightenment style.

These are useful for:
- repo reference pages
- prompt packs
- sticker layouts

- print sheets
- emblem studies
- body-glyph inventories
- fake cult visual systems

## Core principle

A Garbage Enlightenment symbol sheet should not feel like:
- clean UI iconography
- neat vector sticker merch
- spiritually sincere symbol catalog
- random assortment with no family resemblance

It should feel like:
- a symbolic ecosystem
- a contaminated sticker page
- a body-haunted glyph chorus
- a fake revelation inventory
- a family of related signs that all came from the same weird little universe

## SHEET COMPOSITION RULES

### Rule 1

Use repetition with variation.
Motifs should feel related but not cloned.

### Rule 2
Keep one or two dominant motif families.
Do not make every symbol equally unrelated.

### Rule 3
Use enough spacing to preserve readability.
Chaos is allowed, mud is not required.

### Rule 4
At least some symbols should feel mutational or role-unstable.

### Rule 5
A sheet should still have tonal cohesion:
- sweet-heavy
- body-heavy
- fake-sacred
- abstract-emergent
- mixed but intentional

## LSD-LOGIC NOTE

Symbol sheets become more Garbage Enlightenment-specific when:
- multiple symbols imply overlapping roles
- some symbols look like they are almost becoming other symbols
- the set feels over-meaningful and slightly unstable

- repetition creates symbolic pressure rather than mere ornament

That is a very strong static form of cartoonized LSD-hallucination logic.
