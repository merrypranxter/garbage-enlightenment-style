# PROMPT REPAIR GUIDE 02

This file expands on prompt repair with more advanced diagnosis and repair logic.

The first repair guide focuses on common failures.
This one focuses on subtler prompt weakness:
- prompts that are almost right but still dead
- prompts that contain the right nouns but the wrong structure
- prompts that miss the style’s deeper logic

## Core principle

A prompt can mention:
- eyes

- rainbows
- weird creatures
- occult symbols
- psychedelic visuals

and still completely fail to become Garbage Enlightenment.

Why?
Because the real issue is often:
- no body glyph logic
- no contamination logic
- no mutational pressure
- no tonal friction
- no LSD-logic structure
- no anti-polish bias

## ADVANCED FAILURE TYPE 1 — RIGHT INGREDIENTS, WRONG ATTITUDE

### Bad prompt
“A surreal cartoon with eyes, skulls, stars, rainbows, and an occult symbol.”

### Problem
It lists ingredients but gives them no relationship, no tension, and no behavior.

### Repair
Specify:
- symbolic roles
- contamination
- tonal mix
- motion or mutational pressure

### Better version
A low-fi mutant cartoon in the Garbage Enlightenment style where stars and rainbows are
contaminated by eyes and skull motifs, with fake-sacred occult icon staging, rude black outlines,
symbolic body weirdness, and the feeling that the whole image is paused mid-mutation

---

## ADVANCED FAILURE TYPE 2 — TOO NOUN-HEAVY, NOT ENOUGH VERB LOGIC

### Bad prompt
“A face, a rainbow, a hand, a pyramid, and a thermal field.”

### Problem
These are nouns without a logic of becoming.

### Repair
Add metamorphic verbs:
- emerging from

- dissolving into
- contaminated by
- becoming
- implying
- leaking into
- revealing

### Better version
A hand glyph emerging from a thermal contour field while a face leaks out of the hand’s palm
center and a rainbow arc becomes a mouth-like portal beneath it

---

## ADVANCED FAILURE TYPE 3 — TOO DESCRIPTIVE, NOT ENOUGH STRUCTURAL

### Bad prompt
“A colorful psychedelic image with lots of details and weirdness.”

### Problem
It describes appearance vaguely without identifying the style’s actual engines.

### Repair
Name engines:
- body glyphs
- Sweet Corruption

- mock esoterica
- abstract hallucination field
- figure-pattern permeability
- cartoonized LSD-hallucination logic

### Better version
A figure-pattern hybrid in the Garbage Enlightenment style using body glyph symbolism, Sweet
Corruption motifs, mock-esoteric framing, and abstract hallucination fields with cartoonized
LSD-hallucination logic

---

## ADVANCED FAILURE TYPE 4 — TOO STATIC

### Bad prompt
“A hand symbol with an eye in the middle.”

### Problem
Could still work, but risks feeling dead.

### Repair
Add implied motion or mutational instability.

### Better version

A hand glyph with an eye in the palm, rendered as if it were halfway between icon, creature, and
portal, with contour-band aura fields and the sense that the fingers might melt into symbols at
any moment

---

## ADVANCED FAILURE TYPE 5 — LSD NAME DROP WITH NO STRUCTURAL PAYLOAD

### Bad prompt
“LSD style surreal cartoon.”

### Problem
Completely under-specified and lazy.

### Repair
Translate LSD relation into structural qualities:
- symbolic over-meaning
- rapid significance reassignment
- one shape implying another
- pattern/figure instability
- continuous metamorphosis

### Better version
A mutant cartoon image using cartoonized LSD-hallucination logic: symbolic over-meaning,
figure-pattern instability, rapid semantic reassignment, and one form implying several forms at
once, rendered with rude black outlines and body glyph motifs

## REPAIR QUESTIONS

Ask these when a prompt feels almost right but dead:

### 1. What is the body anchor?
If none, add one.

### 2. What is the contamination?
If none, add Sweet Corruption or bodily sabotage.

### 3. What is unstable?
If nothing feels unstable, add mutation pressure.

### 4. What is the tone mixture?
If the tone is singular, add friction.

### 5. What is the fake-sacred or fake-significant element?
If none, add a symbolic charge.

### 6. If abstract, what might emerge from it?
If nothing, it needs pressure.

## FINAL RULE

When repairing a prompt, do not just ask:
“What else can I add?”

Ask:
**“Which of the style’s engines is currently missing or asleep?”**

Wake that engine up.
