# REFERENCE MATCH NOTES

This file explains how to compare new references against the Garbage Enlightenment style.

Useful for:
- sorting image refs
- deciding whether a new piece belongs
- labeling branches
- curating future mood boards
- telling Copilot or other tools how to classify incoming images

## Core principle

A reference does not need to match every trait.
But it should match multiple deep traits, not just surface weirdness.

## HIGH-CONFIDENCE MATCH SIGNALS

A reference is a strong match if it clearly shows several of these:

- rude black outlines
- low-fi cartoon simplification
- body glyph symbolism
- cute contamination / Sweet Corruption
- mock-esoteric or fake-revelation energy
- abstract hallucination field logic
- figure-pattern permeability
- meaningful mutational instability
- anti-polish cheapness
- cartoonized LSD-hallucination structure

## MEDIUM-CONFIDENCE MATCH SIGNALS

A reference may still be useful if it strongly contributes to one subsystem, such as:

### For Sweet Corruption
- stars, hearts, rainbows, cute motifs with weirdness

### For Body Goblin
- awkward little creatures
- pathetic mutant charm
- body-heavy cartoon weirdness

### For Mock Revelation
- eye pyramids
- hands
- aura rings
- parody occult icon staging

### For Abstract Hallucination
- contour bands
- thermal maps
- emergence-friendly psychedelic fields

## LOW-CONFIDENCE MATCH / POSSIBLE FALSE POSITIVES

These may look adjacent but are often not true matches:
- generic trippy AI art
- visionary cosmic paintings
- polished sticker kawaii
- goth occult merch
- abstract psychedelic wallpaper
- designer surreal illustration
- grim monster art
- stylish underground dark graphics with no body weirdness

## REFERENCE TAGGING SUGGESTION

When sorting refs, tag them by:
- overall fit
- subsystem fit
- branch fit
- motion relevance
- palette relevance
- motif relevance

### Example tags
- high-fit
- Sweet Corruption
- body-glyph heavy
- fake revelation
- contour-field
- creature reference
- motion reference
- palette-only
- low-fit neighbor

## REFERENCE QUESTIONS

Ask of each new ref:

### 1. Does it have body glyph intelligence?
### 2. Does it contain contamination or symbolic wrongness?
### 3. Is it too clean or too sincere?
### 4. Does it imply transformation?
### 5. Does it feel fake-sacred, body-weird, or sweetly wrong?
### 6. Is the weirdness specific, or just generic?
### 7. Does it resemble cartoonized LSD-logic in any structural way?

## CLASSIFICATION TIERS

### Tier A — Core reference
Strong enough to help define the style.

### Tier B — Subsystem reference
Useful for one branch or subsystem.

### Tier C — Neighbor reference
Helpful as comparison or contrast, but not direct style DNA.

### Tier D — False friend
Looks related from far away but actually pushes drift.

## FINAL RULE

Do not keep a reference just because it is:
- weird
- psychedelic
- cute
- occult
- gross

Keep it because it strengthens one of the repo’s actual engines without dragging the style
somewhere weaker.
