## TAGGING NOTES

### Use body tags whenever possible
Body logic is one of the style’s strongest anchors.

### Use tone tags actively
Tone matters as much as motif.

### Use motion tags even for stills
If the image implies morphing, say so.

### Use palette tags only when relevant
Do not over-tag color unless it matters to function.

## FINAL RULE

A good tag set should not just say what is visible.

It should also capture:
- what role it plays
- what emotional mode it belongs to
- whether it is cute, bodily, symbolic, mutational, or abstract
- how it behaves inside the style
