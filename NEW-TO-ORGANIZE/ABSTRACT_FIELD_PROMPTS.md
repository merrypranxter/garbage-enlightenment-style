# ABSTRACT FIELD PROMPTS

This file contains prompt formulas for the abstract hallucination field branch of Garbage
Enlightenment.

These prompts are for:
- contour fields
- shader-like topographies
- thermal maps

- plasma blobs
- body-haunted abstractions
- emergence substrates

## Core principle

An abstract field in Garbage Enlightenment should not feel like neutral wallpaper.

It should feel like:
- almost-symbol
- almost-body
- almost-creature
- pressure map for hidden meaning
- fake cosmic weather
- mutational soup

## PROMPT 1 — CONTOUR EMERGENCE FIELD

An abstract hallucination field in the Garbage Enlightenment style made of warped contour
bands, synthetic acid color zones, low-fi signal pressure, and latent body-glyph emergence, as if
eyes, mouths, or hands might resolve from the field at any moment; no generic psychedelic
wallpaper, no majestic aura art, no empty rainbow mush

---

## PROMPT 2 — THERMAL BODY-HAUNTED FIELD

A thermal-rainbow false topology in the Garbage Enlightenment style, with chemically fake
contour transitions, body-adjacent color interruptions, and enough symbolic pressure that the
field feels like a face, mouth, or pathetic creature is trying to form inside it

---

## PROMPT 3 — PLASMA GOBLIN SOUP

A plasma-like abstract field in the Garbage Enlightenment style, built from threshold blobs, liquid
pressure zones, and rude synthetic color, with the sense that blob creatures, eye glyphs, and
little body-signs are repeatedly almost-emerging from the field

---

## PROMPT 4 — SIGNAL SOUP WITH FIGURE PRESSURE

A cheap digital hallucination field in the Garbage Enlightenment style, with contour-band
psychedelia, dithered signal shimmer, symbolic over-meaning, and figure-pattern permeability,
like a body-haunted screen glitch trying to become a warning sign, an eye, or a face

---

## PROMPT 5 — LSD-LOGIC ABSTRACT PRESSURE FIELD

A highly symbolic abstract field in the Garbage Enlightenment style using cartoonized
LSD-hallucination logic: rapid significance reassignment, one contour implying several forms,
figure/ground instability, and body-glyph emergence pressure, all rendered through rude
synthetic cartoon color rather than polished visionary abstraction

## ANTI-DRIFT NOTES

If the abstract field gets too pretty:
- add threshold harshness
- add body implication
- add cheap digital friction
- break gradient smoothness

If it gets too empty:
- name likely emerging glyphs

- add contour pressure
- use stronger symbolic phrasing
- mention face/eye/mouth potential

If it gets too genericly trippy:
- reduce ornamental language
- emphasize emergence
- emphasize body logic
- emphasize anti-polish

## FINAL RULE

An abstract field prompt should feel like:
**a symbolic pressure map in which bodies, creatures, and warnings are trying to be born from
color and contour**
