# MOCK REVELATION PROMPTS

This file contains prompt formulas for the Mock Revelation branch of Garbage Enlightenment.

This is the style zone of:
- eye pyramids
- hand glyphs
- aura rings
- portals
- fake sigils
- revelation posters
- cosmic warning signs
- parody sacred icons

## Core principle

Mock Revelation should feel:
- charged
- symbolic
- slightly convincing
- fake-sacred
- low-fi
- bodily contaminated
- not fully trustworthy

- not fully serious

It should not become:
- clean occult branding
- visionary-art sincerity
- mystical luxury design
- fully reverent symbolism

## ANTI-DRIFT NOTES

If output becomes too sacred:
- add parody
- add body weirdness
- add one sweet or ridiculous contaminant
- break symmetry
- reduce visual nobility

If output becomes too empty:
- strengthen icon staging
- add aura ring or pyramid support
- use stronger symbol hierarchy
- give the image a fake sermon tone

## LSD-LOGIC NOTE

Mock Revelation becomes more specific when it uses **cartoonized LSD-hallucination logic**:
- simple icons feel excessively meaningful
- aura/ring/eye relations feel overcharged
- one symbol implies several symbolic roles at once
- revelation feels chemically vivid but unreliable

That is a useful branch-specific pressure source.

## FINAL RULE

A Mock Revelation prompt should feel like:
**a low-budget cosmic warning from a cartoon reality that cannot decide whether it is divine
truth, body joke, or counterfeit sacred merch**
