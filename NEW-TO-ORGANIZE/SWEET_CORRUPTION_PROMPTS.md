# SWEET CORRUPTION PROMPTS

This file contains prompt formulas specifically for the Sweet Corruption branch of Garbage
Enlightenment.

This is the zone where:
- stars
- hearts
- rainbows
- clouds
- dolphins
- cute little creatures
- sticker logic
- sugary toy symbols

get exposed to:
- bodily weirdness
- symbolic wrongness
- mutation
- fake revelation

- body-glyph contamination

## Core principle

Sweet Corruption is not:
- pure cuteness
- kawaii cleanliness
- harmless sticker art
- fantasy sweetness played straight

It is:
- sweetness made unstable
- cute symbols touched by body logic
- decorative charm contaminated by weirdness
- toy-like imagery with mutational pressure

## SWEET CORRUPTION KEY PHRASES

Useful phrases:
- sweet corruption
- cute but contaminated
- toy-like but wrong
- decorative sweetness touched by body weirdness
- candy symbol with symbolic instability
- sticker charm sabotaged by mutation
- rainbow logic becoming bodily logic
- adorable but anatomically questionable

## ANTI-DRIFT NOTES

If Sweet Corruption output gets too cute:
- add a body glyph
- add one fake-sacred element
- add asymmetry
- add mild grossness
- reduce polish

If it gets too dark:
- restore stars/hearts/clouds
- brighten the sweet motifs
- reduce menace
- keep the weirdness playful

## LSD-LOGIC NOTE

Sweet Corruption becomes especially strong when cute motifs behave with **cartoonized
LSD-hallucination logic**:
- stars nearly becoming eyes
- rainbows nearly becoming mouths
- hearts implying body cavities
- decorative sweetness acting over-significant and symbolically unstable

This is one of the best ways to keep the branch alive and non-generic.

## FINAL RULE

A Sweet Corruption prompt should feel like:
**a cute sticker dream that accidentally learned too much about bodies, symbols, and weird little
revelations**
