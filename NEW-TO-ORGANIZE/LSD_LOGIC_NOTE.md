# LSD LOGIC NOTE

This file clarifies what the repo means when it says Garbage Enlightenment resembles
“LSD-hallucination logic.”

This matters because that phrase can easily get used lazily or stupidly.

## Core principle

When this repo references LSD-like qualities, it is **not** claiming:
- literal documentary accuracy
- medically exact psychedelic depiction
- universal trip realism
- polished visionary-art authenticity
- “just add rainbow and eyeballs”

Instead, it is pointing to a structural resemblance in how images behave.

## WHAT “LSD LOGIC” MEANS HERE

Inside Garbage Enlightenment, LSD-like logic refers to qualities such as:

- rapid significance reassignment
- one form implying multiple forms at once
- symbolic over-meaning
- figure-pattern instability
- recursive implication
- body/cosmos crossover
- visual pressure that feels revelatory even when ridiculous
- continuous metamorphosis instead of stable objecthood
- emotionally charged hue boundaries
- intermediate states that feel unusually alive

## EXAMPLES OF USEFUL LSD-LIKE STRUCTURE

### Example 1
A contour field suddenly reads as a face without fully stopping being a field.

### Example 2
A rainbow becomes a mouth, and the mouth still retains rainbow logic.

### Example 3
An eye feels like a portal, symbol, creature seed, and joke at the same time.

### Example 4
The image feels over-significant, like everything might mean too much at once.

These are the kinds of qualities the repo is talking about.

## WHAT IT DOES NOT MEAN

It does not simply mean:
- more rainbow
- more fractals
- more eyes
- more glowing gradients
- more spiritual cliché
- generic “trippy” art

Those are surface clichés.
The repo is interested in deeper structural behavior.

## WHY THIS LANGUAGE IS STILL USEFUL

Despite the risks, the phrase is useful because it names something real:

the style often has a visual logic where:
- meaning moves too fast
- forms are not stable
- pattern and symbol blur
- significance feels chemically amplified
- the image behaves like a live symbolic event

That resemblance is real and useful, as long as it is handled carefully.

## HOW TO USE IT IN PROMPTS

Use phrases like:
- cartoonized LSD-hallucination logic
- LSD-like symbolic instability
- over-significant morph logic
- pattern-to-figure emergence with hallucinatory pressure
- psychedelic meaning reassignment without polished visionary-art drift

Do **not** rely only on:
- “LSD style”
- “trippy”
- “hallucination art”

Those are too blunt.

## WHY GARBAGE ENLIGHTENMENT IS NOT JUST “LSD STYLE”

Because the repo also depends on:
- body glyph systems
- Sweet Corruption
- mock esoterica
- rude cartoon simplification
- anti-polish cheapness
- pathetic creaturehood
- sticker/emblem readability

Without those, you do not have Garbage Enlightenment.
You just have vague psychedelic reference fog.

## FINAL RULE

If you mention LSD in relation to this style, mean:
**unstable symbolic logic under vivid visual pressure**

Not:
“it has lots of bright colors and weird shit”
