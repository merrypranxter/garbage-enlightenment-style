# BODY PART LIBRARY

This file expands the body-glyph logic into a more practical library for prompt writing, motif
selection, generation systems, and style testing.

The point is not anatomical completeness.
The point is symbolic usability.

## Core principle

In Garbage Enlightenment, body parts are not valuable because they are anatomically correct.
They are valuable because they can function as:
- icons
- emblems
- creatures
- portals
- jokes
- signs
- decorative modules
- revelation props
- mutation seeds

## FINAL RULE

A body part in Garbage Enlightenment should almost never remain only itself.

It should be available to become:
- a sign
- a symbol
- a creature

- a joke
- a portal
- a decorative unit
- an animation event

That is how the body stays alive in the style.
