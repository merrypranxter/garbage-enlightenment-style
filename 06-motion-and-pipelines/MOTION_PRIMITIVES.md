# MOTION PRIMITIVES

This file defines basic motion primitives for Garbage Enlightenment.

These are the small repeatable movement actions that can be used to build loops, transitions,
animations, or generative motion logic.

## Core principle

Motion primitives in this style are usually not ordinary action units.
They are symbolic transformation units.

The style moves through:
- becoming
- leaking
- revealing
- substituting
- pulsing
- dissolving
- reassigning
