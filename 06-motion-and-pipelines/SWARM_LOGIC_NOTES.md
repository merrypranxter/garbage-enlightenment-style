# SWARM LOGIC NOTES

This file explains swarm logic in Garbage Enlightenment.

Swarm logic appears when:
- many small units repeat
- one symbol proliferates
- a face becomes many faces
- blobs subdivide into creature chorus
- decorative repetition starts behaving like distributed consciousness

## Core principle

A swarm in this style is not just “lots of stuff.”
It should feel like:
- repetition with pressure
- a motif chorus
- symbolic infestation
- distributed creaturehood
- decorative consciousness
- fake ritual insistence
- dream bureaucracy made visible

## 4. SYMBOLIC SWARM

### Units
Different symbols repeat under a common logic.

Example:
- eye + heart + mouth + star variants
- all acting like one symbolic family

### Effect
Feels like:
- a living sticker language
- icon ecology
- family resemblance under mutation

## SWARM BEHAVIOR RULES

### Rule 1
A swarm should repeat with variation.
Pure duplication is weak.

### Rule 2
A swarm should preserve motif family resemblance.
Total randomness kills the force.

### Rule 3
A swarm can imply mind or pressure.
It should not only read as decorative wallpaper.

### Rule 4
A swarm can be cute, eerie, gross, or fake-sacred.
But it should remain emotionally active.

## GOOD SWARM QUALITIES

- clustered but readable
- recursive without becoming pattern wallpaper
- slightly too much
- charmingly excessive
- symbolically insistent
- alive through local variation

## BAD SWARM QUALITIES

- perfect uniformity
- meaningless clutter
- decorative filler with no emotional or symbolic charge
- symmetrical mandala drift
- wallpaper prettiness

## SWARM + LSD-LOGIC

Swarm logic can strongly resemble **cartoonized LSD-hallucination structure** when:

- repetition creates over-meaning
- one symbol feels multiplied by significance rather than by logic
- clustered motifs feel conscious
- figure and pattern start to blur under repetition

This is especially strong when the swarm feels:
- a little devotional
- a little dumb
- a little body-made
- a little too emotionally loaded
